## Commit Message

`[wt \d]`をコミットメッセージに含めてください。（`wt`は「Working Time」の略です）  
プルリクがマージまたはクローズされた時、そのプルリクの`base..head`間のコミットのメッセージをすべて読んで`wt`の時間を集計します。そして、`bot`でコメントします。
